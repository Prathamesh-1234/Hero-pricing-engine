# AI Prompts Used During Development

The following prompts were used with AI tools (ChatGPT, Copilot) to assist with brainstorming, code generation, and documentation.

---

## Phase 1 – Problem Understanding
- “Break down the problem statement for the Hero Cycles pricing engine. List the key requirements.”
- “What are the typical entities in a bicycle pricing system? Suggest data models.”
- “What questions should I ask to clarify the scope of this assignment?”

## Phase 2 – Design & Architecture
- “Design a REST API for managing parts, cycles, and quotes with a MongoDB schema.”
- “How should I structure the pricing engine to handle dynamic price changes and snapshots?”
- “Provide the folder structure for a Node.js + React project.”

## Phase 3 – Backend Implementation
- “Write a Mongoose schema for Part with price history.”
- “Generate a controller for part CRUD with price update endpoint.”
- “Create a service that calculates price breakdown for a cycle given its components.”
- “Implement quote creation that stores a snapshot of pricing.”

## Phase 4 – Frontend Implementation
- “Create a React component for Parts management with list, add, and price update.”
- “Build a Cycle configuration component that lets users select parts and quantities.”
- “Implement a Quote generator that shows price preview and creates a quote.”
- “Design a modern, professional CSS style for the entire UI (no gradients).”

## Phase 5 – Testing & Documentation
- “Suggest sample data to seed the database for testing.”
- “Write a comprehensive README.md with setup instructions and API documentation.”
- “List all assumptions I should document for this project.”
- “Generate a list of questions I should have asked while solving this problem.”

## Phase 6 – Commit Strategy
- “Help me plan incremental commits that show progress and each commit should have a meaningful outcome.”

---

All prompts were used to generate code snippets, refine architecture, and ensure the final product meets the assignment’s requirements.
